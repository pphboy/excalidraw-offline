0- 先完成，样式使用最简洁的版本，与 excalidraw 导出的数据无关的，都存localStorage

1. 实现一个excalidraw-main
   - openfile 打开一个 `.excalidraw` 的文件，打开的时候，使用go先把json格式校验一下，如果json格式不对，直接用wails的

   ```
   result, err := runtime.MessageDialog(a.ctx, runtime.MessageDialogOptions{
           Type:          runtime.QuestionDialog,
           Title:         "Question",
           Message:       "Do you want to continue?",
           DefaultButton: "No",
       })
   ```

   进行报错
   - createfile
     开启一个tab, 然后要求用户输入文件名（后缀是强制为 .excalidraw）

   - recents列表
     软件打开之后就是 opencreate,createfile,recent列表
     用于展示最近打开的文件列表，数据存储到localStorage

2. 交互方式
   需要实现一个 main框架(带session 文件tab ， 类似于vscode)，上面有openfile,createfile, autosave(radio)的按钮，然后createfile就新开一个tab， 于tab中打开对应的文件
   tab关闭的时候自动保存

   autosave默认打开

   openFile就是需要先用go调用 wails的文件选择器，然后json值传入 到前端，然后再调用 excalidrawAPI.updateScene()

   如果开启autosave，就通过 onChange事件，（限流2秒触发一次），save到文件中。

   保存文件的快捷键为 Ctrl+S ，执行时就是将 exclidrawAPI的

   ```js
   serializeAsJSON(
        excalidrawAPI.getSceneElements(),
        excalidrawAPI.getAppState(),
        excalidrawAPI.getFiles(),
        "local",
      ),
   ```

   返回的json文件无脑刷入对应的文件中。

   下面的函数是将读取的json无脑刷到一个会话的图中

   ```js
   excalidrawAPI.updateScene();
   ```
